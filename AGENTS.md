**Commands:**
- `cargo build` | `cargo test` | `cargo run`
- `cargo test <test_name>` - single test

**Debugging:**
- Use OpenCode `text-adventure` tool for persistent REPL sessions
- Actions: `start`, `input`, `status`, `list`, `stop`, `kill_server`
- Runs game in `--llm-mode` via stdin/stdout

**Code Style:**
- Rust: snake_case, anyhow::Result, serde JSON
